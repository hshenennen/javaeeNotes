## 📌 题目：移除字符串中的所有空格

**题目描述**：  
编写一个 Java 方法，接收一个字符串，返回一个**完全不含空格**的新字符串（包括开头、中间、结尾的所有空格）。

**示例**：  
- 输入：`"  he llo  world  "`  
- 输出：`"helloworld"`

---

## 💡 思路分析

要移除所有空格，核心是：
- **遍历原字符串的每个字符**
- **跳过空格字符 `' '`，只保留非空格字符**
- **将保留的字符拼接成新字符串**

> ⚠️ 注意：题目中的“空格”通常指普通空格 `' '`（ASCII 32），不包括制表符 `\t`、换行符 `\n` 等。若需处理所有空白字符，需额外说明。

---

## ✅ 方法一：使用 `StringBuilder`（推荐，高效）

```java
public class RemoveSpaces {
    public static String removeAllSpaces(String str) {
        StringBuilder sb = new StringBuilder();
        for (char c : str.toCharArray()) {
            if (c != ' ') {
                sb.append(c);
            }
        }
        return sb.toString();
    }

    public static void main(String[] args) {
        String input = "  he llo  world  ";
        String result = removeAllSpaces(input);
        System.out.println("原字符串: \"" + input + "\"");
        System.out.println("去除空格后: \"" + result + "\"");
        // 输出: "helloworld"
    }
}
```

> ✅ 优点：效率高（`StringBuilder` 可变，避免频繁创建字符串）  
> ✅ 清晰易懂，适合初学者

---

## ✅ 方法二：使用 `String.replace()`

```java
public static String removeAllSpaces(String str) {
    return str.replace(" ", "");
}
```

> ✅ 极其简洁！  
> ⚠️ 注意：`replace(CharSequence target, CharSequence replacement)` 会替换**所有匹配项**，这里正好适用。  
> 🔍 区别：`replaceAll()` 使用正则表达式，此处不需要。

---

## ✅ 方法三：使用正则表达式（扩展，处理所有空白字符）

如果你想移除**所有空白字符**（包括空格、制表符、换行等）：

```java
public static String removeAllWhitespace(String str) {
    return str.replaceAll("\\s+", ""); // \\s 匹配任意空白字符
}
```

> 示例：`"a\t b\n c"` → `"abc"`  
> 📌 初学者可先掌握普通空格处理，正则后续再学

---

## 📚 知识点总结

| 方法 | 说明 |
|------|------|
| `StringBuilder.append()` | 高效拼接字符 |
| `String.replace(" ", "")` | 快速替换所有空格（简单场景首选） |
| `String.replaceAll("\\s+", "")` | 使用正则移除所有空白字符 |
| `toCharArray()` | 将字符串转为字符数组，便于逐个检查 |
| 空格 `' '` vs 空白字符 `\s` | 普通空格只是空白字符的一种 |

## 🧠 小练习建议

1. **基础练习**：用 `replace()` 和 `StringBuilder` 两种方式实现，并比较代码简洁性和可读性。
2. **边界测试**：
   - 输入为空字符串 `""`
   - 输入全是空格 `"   "`
   - 输入无空格 `"hello"`
3. **扩展挑战**：
   - 编写方法只移除**首尾空格**（提示：用 `trim()`）
   - 编写方法将**多个连续空格**替换为**单个空格**（如 `"a   b"` → `"a b"`）


---

💡 **小贴士**：在实际开发中，如果只是简单去空格，`str.replace(" ", "")` 是最简洁高效的选择；如果需要更复杂的逻辑（比如过滤其他字符），再考虑 `StringBuilder` 遍历。
