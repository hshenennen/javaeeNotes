## 📌 题目：反转字符串

**题目描述**：  
编写一个 Java 程序，将输入的字符串反转后输出。

---

## 💡 思路分析

Java 中有多种方式可以实现字符串反转，常见的方法包括：

1. **使用 `StringBuilder` 或 `StringBuffer` 的 `reverse()` 方法**（最简单）
2. **手动遍历字符串，从后往前拼接字符**
3. **使用递归（进阶）**

作为初学者，建议先掌握第 1 和第 2 种方法。

---

## ✅ 方法一：使用 `StringBuilder`

```java
public class ReverseString {
    public static void main(String[] args) {
        String original = "Hello, Java!";
        StringBuilder sb = new StringBuilder(original);
        String reversed = sb.reverse().toString();
        System.out.println("原字符串: " + original);
        System.out.println("反转后: " + reversed);
    }
}
```

> ✅ 优点：代码简洁，效率高（`StringBuilder` 是可变字符串）  
> ⚠️ 注意：`String` 本身是不可变的，所以不能直接修改

---

## ✅ 方法二：手动遍历反转

```java
public class ReverseStringManual {
    public static void main(String[] args) {
        String original = "Hello, Java!";
        String reversed = "";
        
        for (int i = original.length() - 1; i >= 0; i--) {
            reversed += original.charAt(i);
        }
        
        System.out.println("原字符串: " + original);
        System.out.println("反转后: " + reversed);
    }
}
```

> ⚠️ 注意：这种方式在循环中频繁拼接字符串，效率较低（因为每次 `+=` 都会创建新对象）  
> 🔜 后续可以改用 `StringBuilder` 在循环中拼接来优化

---

## 📚 知识点总结

| 概念 | 说明 |
|------|------|
| `String` | 不可变类，一旦创建内容不能更改 |
| `StringBuilder` | 可变字符串，适合频繁修改字符串内容 |
| `charAt(int index)` | 获取字符串指定位置的字符（从 0 开始） |
| `length()` | 获取字符串长度 |
| `reverse()` | `StringBuilder`/`StringBuffer` 提供的反转方法 |
