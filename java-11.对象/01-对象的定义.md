## 📌 对象的定义（Java 面向对象编程）

### ✅ 核心定义
> **对象（Object）** 是 **类（Class）的一个具体实例**，用于表示现实世界中的某个实体，具有 **状态（属性）** 和 **行为（方法）**。

在 Java 中，一切皆对象（除了基本数据类型），程序通过对象之间的交互来完成任务。

---

### 🧠 类比理解
| 概念 | 类比 | 说明 |
|------|------|------|
| **类（Class）** | 蛋糕模具 | 描述“蛋糕”应该长什么样（有几层、什么口味） |
| **对象（Object）** | 用模具做出的具体蛋糕 | 实际存在的一个蛋糕，比如“巧克力三层蛋糕” |

> 🔸 没有类，就无法创建对象；  
> 🔸 一个类可以创建多个对象（实例）。

---

### 📦 对象的组成

1. **状态（State）**  
   - 由 **字段（成员变量）** 表示  
   - 描述对象当前的特征或数据  
   - 例如：`Student` 对象的 `name = "张三"`、`age = 20`

2. **行为（Behavior）**  
   - 由 **方法（成员函数）** 表示  
   - 描述对象能做什么  
   - 例如：`student.study()`、`car.startEngine()`

3. **标识（Identity）**（隐式）  
   - 每个对象在内存中有唯一地址（即使内容相同，也是不同对象）  
   - Java 中可通过 `==` 或 `hashCode()` 间接体现

---

### 💡 Java 示例

```java
class Dog {
    // 状态（属性）
    String name;
    int age;

    // 行为（方法）
    void bark() {
        System.out.println(name + " 汪汪叫！");
    }
}

public class Main {
    public static void main(String[] args) {
        // 创建对象（实例化）
        Dog myDog = new Dog();
        myDog.name = "旺财";
        myDog.age = 3;
        myDog.bark(); // 输出：旺财 汪汪叫！
    }
}
```

> ✅ `myDog` 就是一个 **对象**，它是 `Dog` 类的一个实例。
