## 📌 成员变量与成员方法

> 在 Java 中，**类（Class）** 是对象的模板，而 **成员变量** 和 **成员方法** 是构成类的两大核心组成部分，共同描述对象的**状态**与**行为**。

---

### 1️⃣ 成员变量（Member Variables）

#### ✅ 定义
> 成员变量是定义在**类中、方法外**的变量，用于存储对象的状态（属性）。

```java
public class Student {
    // 成员变量（实例变量）
    private String name;   // 姓名
    private int age;       // 年龄
    public static int count = 0; // 静态成员变量（属于类）
}
```

#### 🔑 类型分类

| 类型 | 说明 | 生命周期 | 存储位置 |
|------|------|--------|--------|
| **实例变量** | 无 `static` 修饰 | 对象创建 → 对象销毁 | 堆（Heap） |
| **静态变量** | 有 `static` 修饰 | 类加载 → 程序结束 | 方法区（Method Area） |

> 💡 默认值规则（未显式初始化时）：
> - 数值类型：`0` / `0.0`
> - boolean：`false`
> - 引用类型：`null`

#### 📌 特点
- 属于对象（实例变量）或类（静态变量）
- 可被访问修饰符控制（`private`/`public`/`protected`/`default`）
- 实例变量每个对象独立一份，静态变量所有对象共享一份

---

### 2️⃣ 成员方法（Member Methods）

#### ✅ 定义
> 成员方法是定义在**类中**的函数，用于描述对象的行为或操作数据。

```java
public class Student {
    private String name;

    // 成员方法（实例方法）
    public void study() {
        System.out.println(name + " 正在学习");
    }

    // 静态成员方法
    public static void printSchool() {
        System.out.println("清华大学");
    }
}
```

#### 🔑 类型分类

| 类型 | 说明 | 能否访问实例成员 | 调用方式 |
|------|------|----------------|--------|
| **实例方法** | 无 `static` 修饰 | ✅ 可以（有 `this`） | `对象.方法()` |
| **静态方法** | 有 `static` 修饰 | ❌ 不能（无 `this`） | `类名.方法()` |

#### 📌 特点
- 方法内部可访问类的成员变量和其他方法
- 实例方法隐含 `this` 引用，指向当前对象
- 静态方法不能直接访问非静态成员（需通过对象）

---

### 🔁 成员变量 vs 成员方法 对比表

| 特性 | 成员变量 | 成员方法 |
|------|--------|--------|
| **作用** | 描述对象的**状态**（What it has） | 描述对象的**行为**（What it does） |
| **定义位置** | 类中、方法外 | 类中 |
| **默认值** | 有（如 `0`, `null`） | 无（必须显式定义逻辑） |
| **调用方式** | 通过对象或类（静态）访问 | 通过对象或类（静态）调用 |
| **内存分配** | 实例变量在堆，静态变量在方法区 | 方法代码在方法区，调用时在栈 |

---

### 💡 示例：完整类结构

```java
public class Car {
    // 成员变量（实例）
    private String brand;
    private double speed;

    // 静态成员变量
    public static final String TYPE = "Automobile";

    // 构造方法（特殊成员方法）
    public Car(String brand) {
        this.brand = brand;
    }

    // 实例方法（成员方法）
    public void accelerate() {
        this.speed += 10;
        System.out.println(brand + " 加速到 " + speed + " km/h");
    }

    public double getSpeed() {
        return speed;
    }

    // 静态方法
    public static void showType() {
        System.out.println("车辆类型：" + TYPE);
    }
}

// 使用
Car c = new Car("Tesla");
c.accelerate();          // 调用实例方法
Car.showType();          // 调用静态方法
System.out.println(c.getSpeed()); // 访问实例变量（通过 getter）
```

### ⚠️ 常见误区

| 误区 | 正确理解 |
|------|--------|
| “成员变量就是局部变量” | ❌ 局部变量在方法内，生命周期短；成员变量属于类/对象 |
| “静态方法可以访问实例变量” | ❌ 必须通过对象引用（如 `obj.name`） |
| “成员方法必须有返回值” | ❌ 可以是 `void`（无返回值） |
| “成员变量不能是 final” | ❌ 可以是 `final`（如不可变对象字段） |
