## ✅ 各章节预习指南

### 🔹 [[01-Hello World]]

> [!tip] 学习目标
> - 写出第一个 Java 程序
> - 理解 `public class`、`main` 方法、`System.out.println`
> - 掌握编译与运行流程（`javac` → `java`）

#### 核心知识点：
- `class HelloWorld { ... }`：定义一个类
- `public static void main(String[] args)`：程序入口
- `System.out.println("Hello World!");`：控制台输出

#### 常见问题：
- 编译报错：`cannot find symbol` → 类名与文件名不一致
- 运行失败：`Exception in thread "main"` → 主方法写错

#### 推荐练习：
```java
// 修改输出内容，如打印你的名字
System.out.println("你好，我是张三！");
```

### 🔹 [[02-注释]]

> [!tip] 学习目标
> - 掌握三种注释方式
> - 理解注释的作用：提高代码可读性

#### 核心知识点：
| 注释类型 | 语法 | 用途 |
|---------|------|------|
| 单行注释 | `// 这是注释` | 一行说明 |
| 多行注释 | `/* ... */` | 多行说明 |
| 文档注释 | `/** ... */` | 生成 API 文档 |

#### 常见问题：
- 忘记闭合多行注释 → 代码无法编译
- 注释太多影响阅读 → 适度使用

#### 推荐练习：
在 `HelloWorld.java` 中添加注释说明每行代码作用。

### 🔹[[ 03-对象和类的基本认识]]

> [!tip] 学习目标
> - 理解“类”是模板，“对象”是实例
> - 能区分类与对象的概念

#### 核心知识点：
- **类（Class）**：描述事物的属性和行为（如 `Student` 类）
- **对象（Object）**：类的具体实例（如 `小明` 是 `Student` 对象）
- 示例：
```java
class Student {
    String name;
    int age;
}
Student xiaoMing = new Student(); // 创建对象
```

#### 常见问题：
- 混淆“类”和“对象”
- 不理解 `new` 的作用

#### 推荐练习：
画一个类图：`Person` 类有 `name`, `age` 属性，创建两个对象。

### 🔹[[04-基本数据类型]]

> [!tip] 学习目标
> - 掌握 8 种基本数据类型及其大小
> - 理解 `int`, `double`, `char`, `boolean` 的用途

#### 核心知识点：
| 类型 | 大小 | 示例 |
|------|------|------|
| `byte` | 1 字节 | `-128 ~ 127` |
| `short` | 2 字节 | `-32768 ~ 32767` |
| `int` | 4 字节 | `-21亿 ~ 21亿` |
| `long` | 8 字节 | `10000000000L` |
| `float` | 4 字节 | `3.14f` |
| `double` | 8 字节 | `3.1415926` |
| `char` | 2 字节 | `'A'`, `'中'` |
| `boolean` | 1 字节 | `true`, `false` |

#### 常见问题：
- `int` 和 `long` 混用导致溢出
- `float` 精度不够 → 金融计算用 `double` 或 `BigDecimal`

#### 推荐练习：
声明并赋值不同类型的变量：
```java
int a = 100;
double b = 3.14;
char c = 'X';
boolean flag = true;
```

### 🔹 [[05-变量]]

> [!tip] 学习目标
> - 理解变量是“存储数据的容器”
> - 掌握声明与赋值语法

#### 核心知识点：
- 声明格式：`数据类型 变量名;`
- 赋值：`变量名 = 值;`
- 可以同时声明多个同类型变量：
```java
int a, b, c;
a = 1; b = 2; c = 3;
```

#### 常见问题：
- 使用未初始化的变量 → 编译错误
- 数据类型不匹配 → 类型转换错误

#### 推荐练习：
声明一个 `double` 类型变量表示身高，并赋值。

### 🔹[[06-变量命名规则]]

> [!tip] 学习目标
> - 掌握合法的变量命名规范
> - 避免命名错误

#### 核心知识点：
| 规则 | 说明 |
|------|------|
| 以字母、`_`、`$` 开头 | 不能以数字开头 |
| 只能包含字母、数字、`_`、`$` | 不能有空格或特殊符号 |
| 区分大小写 | `age` 和 `Age` 是不同变量 |
| 不能是关键字 | 如 `int`, `class`, `public` |
| 推荐驼峰命名法 | `studentName`, `totalScore` |

#### 常见问题：
- 使用中文变量名 → 不推荐
- 命名过短（如 `a`, `x`）→ 不易维护

#### 推荐练习：
给以下变量起名：
- 学生姓名 → `studentName`
- 年龄 → `age`
- 是否通过考试 → `passed`

### 🔹 [[07-运算符]]

> [!tip] 学习目标
> - 掌握算术、赋值、比较、逻辑运算符
> - 理解 `++`, `--`, `+=` 等复合运算符

#### 核心知识点：
| 类型 | 运算符 | 示例 |
|------|--------|------|
| 算术 | `+`, `-`, `*`, `/`, `%` | `5 % 2 = 1` |
| 赋值 | `=`, `+=`, `-=` | `a += 1` 等价于 `a = a + 1` |
| 比较 | `==`, `!=`, `<`, `>`, `<=`, `>=` | `5 == 3` → `false` |
| 逻辑 | `&&`, `||`, `!` | `true && false` → `false` |

#### 常见问题：
- `=` 与 `==` 混淆 → `if (a = 5)` 是赋值，不是判断
- `&&` 与 `||` 优先级混乱

#### 推荐练习：
```java
int a = 10, b = 3;
System.out.println(a / b); // 输出 3
System.out.println(a % b); // 输出 1
```

### 🔹[[08-运算符优先级]]

> [!tip] 学习目标
> - 理解运算符执行顺序
> - 掌握括号的使用

#### 核心知识点：
- 优先级从高到低：
  ```
  () → * / % → + - → = += -=
  ```
- 使用括号明确优先级，避免歧义

#### 常见问题：
- `a + b * c` 被误认为 `(a + b) * c`
- 忽略括号导致逻辑错误

#### 推荐练习：
计算表达式：`5 + 3 * 2 - 1` → 结果是 `10`

### 🔹[[09-类型转换]]

> [!tip] 学习目标
> - 掌握自动类型提升与强制类型转换
> - 理解精度丢失风险

#### 核心知识点：
| 类型   | 说明                            |                    |
| ---- | ----------------------------- | ------------------ |
| 自动提升 | 小类型 → 大类型（如 `int` → `double`） |                    |
| 强制转换 | 大类型 → 小类型（如 `double` → `int`） |                    |
| 语法   | `(目标类型) 值`                    | `(int) 3.14` → `3` |

#### 常见问题：
- 强制转换导致数据截断（如 `double` 转 `int` 丢小数）
- `String` 不能直接转 `int` → 需用 `Integer.parseInt()`

#### 推荐练习：
```java
double d = 3.9;
int i = (int) d; // i = 3
```

### 🔹[[10-打印（输出）语句]]

> [!tip] 学习目标
> - 掌握 `System.out.print()` 与 `println()`
> - 学会使用 `printf()` 格式化输出

#### 核心知识点：
- `println()`：输出后换行
- `print()`：输出不换行
- `printf()`：支持格式化（如 `%.2f`）

#### 推荐练习：
```java
System.out.printf("圆周率: %.2f\n", 3.14159);
```

编写程序：输入姓名和年龄，输出问候语。
