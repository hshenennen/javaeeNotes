# 什么是 Java  
- Java 是由 Sun Microsystems 公司于 1995 年 5 月推出的面向对象程序设计语言及平台。  
- 后来 Sun 被 Oracle Corporation 收购，Java 成为 Oracle 的产品。  
- Java 分为三个主要体系：  
  - Java SE（原 J2SE）  
  - Java EE（原 J2EE）  
  - Java ME（原 J2ME）  
- 在 2005 年 6 月的 JavaOne 大会上，Sun 宣布 Java SE 6 并废弃版本命名前的 “2”。
