# Java 的主要特性  
以下是 Java 语言的核心特性：  
- **简单**：语法与 C／C++ 相似，去除复杂特性。  
- **面向对象**：类、接口、继承、动态绑定。  
- **分布式**：内置网络编程 API、支持远程方法调用。  
- **健壮**：强类型、异常处理、自动垃圾回收。  
- **安全**：类加载器、字节码校验、安全管理机制。  
- **体系结构中立**：编译为字节码，可在多平台运行。  
- **可移植**：与“体系结构中立”相辅相成，数据类型长度固定。  
- **解释型**：编译为字节码，在 JVM 上运行。  
- **高性能**：随着即时编译（JIT）技术，性能提升。  
- **多线程**：内置线程支持，同步机制（`synchronized`）等。  
- **动态**：类可在运行时加载、具备运行时类型信息。
