# Java 对象与类（面向对象编程基础）

> [!info] 学习目标
> - 理解什么是“类”和“对象”
> - 掌握如何定义类和创建对象
> - 了解成员变量与方法
> - 初步认识 `new` 关键字和构造方法

---

## 1. 什么是类（Class）？

- **类** 是一种**模板**或**蓝图**，用于描述一类事物的共同特征（属性）和行为（方法）。
- 它本身不占用内存，只有在创建对象时才会分配内存。

### 🌰 举例：汽车（Car）
- 所有汽车都有：品牌（brand）、颜色（color）、速度（speed） → **属性（字段）**
- 所有汽车都能：启动（start）、加速（accelerate） → **行为（方法）**

```java
public class Car {
    // 成员变量（属性）
    String brand;
    String color;
    int speed;

    // 方法（行为）
    void start() {
        System.out.println(brand + " 启动了！");
    }

    void accelerate(int increment) {
        speed += increment;
        System.out.println("当前速度: " + speed + " km/h");
    }
}
```

---

## 2. 什么是对象（Object）？

- **对象** 是**类的具体实例**。
- 你可以根据同一个类创建多个不同的对象。

### 🌰 创建 Car 对象

```java
public class Main {
    public static void main(String[] args) {
        // 使用 new 创建对象
        Car myCar = new Car();

        // 给属性赋值
        myCar.brand = "Toyota";
        myCar.color = "红色";
        myCar.speed = 0;

        // 调用方法
        myCar.start();           // 输出：Toyota 启动了！
        myCar.accelerate(30);    // 输出：当前速度: 30 km/h
    }
}
```

> ✅ `myCar` 是 `Car` 类的一个**对象（实例）**。

## 3. 构造方法（Constructor）

- 构造方法用于**初始化对象**。
- 名称必须与类名相同，没有返回类型（连 `void` 都不能写）。
- 如果你没写构造方法，Java 会提供一个**默认无参构造方法**。

### 自定义构造方法

```java
public class Car {
    String brand;
    String color;
    int speed;

    // 有参构造方法
    public Car(String brand, String color) {
        this.brand = brand;
        this.color = color;
        this.speed = 0; // 默认速度为0
    }

    // ... 其他方法
}
```

使用构造方法创建对象：

```java
Car car1 = new Car("BMW", "黑色");
car1.start(); // BMW 启动了！
```

> 💡 `this` 关键字表示当前对象，用于区分成员变量和参数名。

---

## 4. 类 vs 对象 对比表

| 项目 | 类（Class） | 对象（Object） |
|------|-------------|----------------|
| 性质 | 抽象的模板 | 具体的实例 |
| 内存 | 不占用运行时内存 | 占用堆内存 |
| 创建方式 | 用 `class` 关键字定义 | 用 `new` 关键字创建 |
| 举例 | “汽车”这个概念 | 你的那辆红色丰田车 |

---

## 5. 常见误区提醒

> [!warning] 注意
> - 类名首字母大写（如 `Car`, `Student`），遵循 PascalCase。
> - 对象变量存储的是**引用**（内存地址），不是对象本身。
> - 没有 `new` 就没有对象！`Car c;` 只是声明了一个引用，未创建对象。
