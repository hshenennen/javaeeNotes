# Java 基本数据类型（Primitive Data Types）

> [!info] 学习目标
> - 掌握 Java 的 8 种基本数据类型
> - 理解每种类型的用途、取值范围和默认值
> - 学会如何声明和使用变量
> - 区分基本类型与引用类型（初步）

---

## 1. 为什么需要数据类型？

在 Java 中，**所有变量必须先声明类型**，因为 Java 是**强类型语言**。  
数据类型决定了：
- 变量能存储什么值
- 占用多少内存
- 能进行哪些操作

---

## 2. Java 的 8 种基本数据类型

Java 的基本数据类型分为四类：**整数、浮点数、字符、布尔值**。

| 类型 | 关键字 | 占用内存 | 取值范围 | 默认值 | 用途示例 |
|------|--------|----------|----------|--------|----------|
| 字节型 | `byte` | 1 字节 | -128 到 127 | `0` | 小整数，节省内存 |
| 短整型 | `short` | 2 字节 | -32,768 到 32,767 | `0` | 较小整数 |
| 整型 | `int` | 4 字节 | -2³¹ 到 2³¹-1（约 ±21 亿） | `0` | **最常用整数类型** |
| 长整型 | `long` | 8 字节 | -2⁶³ 到 2⁶³-1 | `0L` | 超大整数（如时间戳） |
| 单精度浮点 | `float` | 4 字节 | 约 ±3.4×10³⁸（6-7 位有效数字） | `0.0f` | 小数（需加 `f` 后缀） |
| 双精度浮点 | `double` | 8 字节 | 约 ±1.7×10³⁰⁸（15 位有效数字） | `0.0` | **最常用小数类型** |
| 字符型 | `char` | 2 字节 | 0 到 65,535（Unicode 字符） | `'\u0000'` | 单个字符（用单引号） |
| 布尔型 | `boolean` | JVM 决定 | `true` 或 `false` | `false` | 条件判断 |

> ✅ **初学者建议**：优先使用 `int`、`double`、`char`、`boolean`，其他类型按需使用。

---

## 3. 声明与初始化变量

```java
// 整数
int age = 25;
long population = 7_800_000_000L; // 注意 L 后缀

// 小数
double price = 19.99;
float temperature = 36.5f; // 必须加 f

// 字符
char grade = 'A'; // 单引号！

// 布尔值
boolean isJavaFun = true;
```

> [!warning] 注意
> - `float` 字面量必须加 `f` 或 `F`，否则默认是 `double`。
> - `char` 使用 **单引号** `'A'`，字符串（String）才用双引号 `"Hello"`（String 是引用类型，不是基本类型！）。

---

## 4. 默认值（仅适用于类的成员变量）

```java
public class Example {
    byte b;     // 0
    short s;    // 0
    int i;      // 0
    long l;     // 0L
    float f;    // 0.0f
    double d;   // 0.0
    char c;     // '\u0000'（空字符）
    boolean flag; // false
}
```

> ⚠️ **局部变量（方法内）没有默认值**！必须手动初始化，否则编译报错。

---

## 5. 类型字面量后缀

| 类型 | 字面量写法 | 示例 |
|------|-----------|------|
| `long` | 数字 + `L` 或 `l` | `100L` |
| `float` | 数字 + `F` 或 `f` | `3.14f` |
| `double` | 可选 `D` 或 `d`（默认就是 double） | `2.5` 或 `2.5d` |

## 6. 小练习

尝试运行以下代码，观察输出：

```java
public class DataTypesDemo {
    public static void main(String[] args) {
        int x = 10;
        double y = 3.14;
        char symbol = '$';
        boolean passed = true;

        System.out.println("x = " + x);
        System.out.println("y = " + y);
        System.out.println("symbol = " + symbol);
        System.out.println("passed = " + passed);
    }
}
```

---

## 7. 常见误区

> [!danger] 初学者易错点
> - ❌ `char c = "A";` → 错！应该用单引号：`'A'`
> - ❌ `float f = 3.14;` → 错！应写为 `3.14f`
> - ❌ 把 `String` 当成基本类型 → `String` 是**引用类型**（后面会学）
> - ❌ 在方法里使用未初始化的局部变量 → 编译错误！

---

## 8. 扩展：基本类型 vs 引用类型

| 对比项 | 基本类型（Primitive） | 引用类型（Reference） |
|--------|------------------------|------------------------|
| 存储内容 | 实际值（如 10, 'A'） | 对象的内存地址 |
| 默认值 | 有（如 0, false） | `null` |
| 示例 | `int`, `char`, `boolean` | `String`, `Scanner`, 自定义类 |
| 内存位置 | 栈（Stack） | 对象在堆（Heap），引用在栈 |

> 🔜 后续你会学到：`Integer`, `Double` 等是基本类型的“包装类”，属于引用类型。
