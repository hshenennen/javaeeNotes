# ➕ 用户输入两个数字并相加

> [!info] 学习目标
> - 使用 `Scanner` 获取用户输入  
> - 实现两个整数（或小数）的加法  
> - 理解基本输入输出流程  


> 🏷️ 标签：`#java/基础` `#Scanner` `#用户输入` `#编程/练习`


---

## ✅ 完整代码（整数版）

```java
import java.util.Scanner;

public class AddTwoNumbers {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("请输入第一个整数: ");
        int num1 = scanner.nextInt();

        System.out.print("请输入第二个整数: ");
        int num2 = scanner.nextInt();

        int sum = num1 + num2;
        System.out.println("两数之和为: " + sum);

        scanner.close();
    }
}
```

### 🖨️ 示例运行（整数）
```
请输入第一个整数: 15
请输入第二个整数: 28
两数之和为: 43
```

## ✅ 完整代码（支持小数版）

如果你希望支持小数（如 `3.14 + 2.5`），只需将类型改为 `double`：

```java
import java.util.Scanner;

public class AddTwoNumbersDouble {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("请输入第一个数字: ");
        double num1 = scanner.nextDouble();

        System.out.print("请输入第二个数字: ");
        double num2 = scanner.nextDouble();

        double sum = num1 + num2;
        System.out.println("两数之和为: " + sum);

        scanner.close();
    }
}
```

### 🖨️ 示例运行（小数）
```
请输入第一个数字: 12.5
请输入第二个数字: 7.8
两数之和为: 20.3
```

## 🔍 核心知识点

| 组件 | 说明 |
|------|------|
| `import java.util.Scanner;` | 导入 Scanner 类（必须） |
| `Scanner scanner = new Scanner(System.in);` | 创建 Scanner 对象，读取控制台输入 |
| `scanner.nextInt()` | 读取一个整数（输入非整数会报错） |
| `scanner.nextDouble()` | 读取一个双精度小数 |
| `scanner.close();` | 关闭资源，良好编程习惯 |

---

## ⚠️ 注意事项与常见错误

| 问题 | 说明 | 解决方案 |
|------|------|--------|
| 输入非数字（如字母） | 程序崩溃（`InputMismatchException`） | 初学可忽略；进阶可用 `try-catch` 或 `hasNextInt()` 校验 |
| 忘记导入 `Scanner` | 编译错误 | 添加 `import java.util.Scanner;` |
| 忘记关闭 `scanner` | 资源泄露（小项目影响不大） | 养成 `close()` 习惯 |
| 混淆 `nextLine()` 与其他 `nextXxx()` | 输入错位 | 本例无需 `nextLine()`，保持简单即可 |

> 💡 **初学者建议**：先确保输入合法（如只输数字），后续再学异常处理。
