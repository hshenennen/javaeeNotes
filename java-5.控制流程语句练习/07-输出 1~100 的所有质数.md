# 🔢 输出 1~100 的所有质数

> [!info] 学习目标
> - 综合运用 `for` 循环与条件判断  
> - 复用质数判断逻辑（嵌套循环）  
> - 输出格式美观（每行显示若干个质数）  
> - **不使用任何自定义方法**，全部逻辑在 `main` 中完成  
> - 理解效率优化（√n 剪枝）

> 🏷️ 标签：`#java/循环` `#算法/质数` `#编程/练习` `#嵌套循环`

## ✅ 完整代码（无方法版）

```java
import java.util.Scanner;

public class PrintPrimes1To100 {
    public static void main(String[] args) {
        System.out.println("1 到 100 之间的质数有：");

        int count = 0; // 用于控制每行输出个数

        // 遍历 2 到 100（1 不是质数，跳过）
        for (int n = 2; n <= 100; n++) {
            boolean isPrime = true;

            // 特殊处理：2 是质数
            if (n == 2) {
                // 直接保留 isPrime = true
            }
            // 排除偶数（除2外）
            else if (n % 2 == 0) {
                isPrime = false;
            }
            // 检查奇数因数（从3到√n）
            else {
                int limit = (int) Math.sqrt(n);
                for (int i = 3; i <= limit; i += 2) {
                    if (n % i == 0) {
                        isPrime = false;
                        break;
                    }
                }
            }

            // 如果是质数，输出
            if (isPrime) {
                System.out.print(n + "\t"); // 用制表符对齐
                count++;
                // 每行输出 10 个质数后换行
                if (count % 10 == 0) {
                    System.out.println();
                }
            }
        }

        // 如果最后一行不满10个，手动换行
        if (count % 10 != 0) {
            System.out.println();
        }

        System.out.println("\n共找到 " + count + " 个质数。");
    }
}
```

## 🔍 逻辑解析

### 1. 外层循环：遍历 2~100
- 从 `2` 开始（因为 1 不是质数）
- 对每个 `n` 判断是否为质数

### 2. 内层判断逻辑（复用质数规则）：
- `n == 2` → 质数 ✅  
- `n % 2 == 0` → 非质数 ❌  
- 否则：检查 `3 ~ √n` 的奇数是否能整除 `n`

### 3. 输出格式控制
- 使用 `\t`（制表符）对齐数字
- 每输出 10 个换一行，提升可读性

---

## ⚙️ 优化说明

| 优化点 | 效果 |
|-------|------|
| 跳过偶数（除2） | 减少约 50% 的内层循环 |
| 循环到 `√n` | 将最坏时间复杂度从 O(n²) 降到 O(n√n) |
| 使用 `break` | 找到因数立即退出，避免无效计算 |

> 💡 对于 `n=100`，最大只需检查到 `i=10`，非常高效！

## ⚠️ 常见问题

| 问题 | 解决方案 |
|------|--------|
| 把 1 当成质数 | 明确从 `n=2` 开始循环 |
| 输出格式混乱 | 使用 `\t` 或固定宽度（如 `%3d`） |
| 忘记 `break` | 导致非质数也被误判 |
| 内层循环从 2 开始 | 可以，但效率低；我们已排除偶数，所以从 3 开始更优 |
