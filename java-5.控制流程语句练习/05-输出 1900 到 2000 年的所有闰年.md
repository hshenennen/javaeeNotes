# 📅 输出 1900 到 2000 年的所有闰年

> [!info] 学习目标
> - 复用闰年判断规则  
> - 使用 `for` 循环遍历年份范围  
> - 格式化输出（每行显示若干年份）  
> - **所有逻辑在 `main` 方法中完成，不定义额外方法**  
> - 巩固复合条件表达式的应用

> 🏷️ 标签：`#java/循环` `#条件语句` `#日期处理` `#编程/练习`

---

## ✅ 完整代码

```java
public class LeapYears1900To2000 {
    public static void main(String[] args) {
        System.out.println("1900 年到 2000 年之间的闰年有：");

        int count = 0; // 用于控制每行输出个数

        for (int year = 1900; year <= 2000; year++) {
            // 闰年判断规则：
            // 能被4整除且不能被100整除，或者能被400整除
            if ((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)) {
                System.out.print(year + "\t"); // 制表符对齐
                count++;
                // 每行输出 10 个年份后换行
                if (count % 10 == 0) {
                    System.out.println();
                }
            }
        }

        // 如果最后一行不满10个，手动换行
        if (count % 10 != 0) {
            System.out.println();
        }

        System.out.println("\n共找到 " + count + " 个闰年。");
    }
}
```

## 🔍 闰年规则回顾

一个年份是闰年，当且仅当满足：

```java
(year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)
```

### 关键年份验证：
| 年份 | 是否闰年 | 原因 |
|------|--------|------|
| 1900 | ❌ 否 | 能被100整除，但不能被400整除 |
| 1904 | ✅ 是 | 能被4整除，且不是整百年 |
| 2000 | ✅ 是 | 能被400整除 |

> 💡 因此，1900 **不会**出现在输出中，而 2000 **会**。

## ⚠️ 常见错误提醒

| 错误 | 后果 | 修正 |
|------|------|------|
| 忘记排除 1900 | 多输出一个错误年份 | 使用完整闰年公式 |
| 写成 `year % 4 == 0` | 1900 被误判为闰年 | 必须加上 `%100` 和 `%400` 条件 |
| 循环写成 `year < 2000` | 漏掉 2000 年 | 应为 `year <= 2000` |
