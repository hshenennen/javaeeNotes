# Java 发展历史  
下面按时间顺序列出 Java 的主要里程碑：  
- 1995 年 5 月：Java 语言诞生。  
- 1996 年 1 月：第一个 JDK（JDK 1.0）发布。  
- 1996 年 4 月：10 个主要操作系统供应商宣布将嵌入 Java 技术。  
- 1996 年 9 月：约 8.3 万个网页使用 Java 技术。  
- 1997 年 2 月 18 日：JDK 1.1 发布。  
- …（中间略）  
- 2009 年 4 月 20 日：Oracle 以 74 亿美元收购 Sun，取得 Java 版权。  
- 2011 年 7 月 28 日：Java 7.0 正式版发布。  
- 2014 年 3 月 18 日：Java SE 8 发布。  
- 2017 年 9 月 21 日：Java SE 9 发布。
