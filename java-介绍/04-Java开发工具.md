# Java 开发工具  
开发 Java 程序时常用的环境及工具包括：  
- 操作系统：Linux、macOS、Windows 等。  
- 安装 JDK（如 Java JDK 7、8…）  
- 编辑器或其他工具：如 VS Code 等。  
- IDE（集成开发环境）：如 Eclipse、IntelliJ IDEA、NetBeans 等。  
- 创建并运行一个简单 Java 程序，例如：
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```
